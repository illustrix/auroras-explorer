# Auroras Explorer

A community web tools collection for [Prosperous Universe](https://prosperousuniverse.com/)

Built to help players explore game data, plan production, and collaborate with others.

[![MIT License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Bun](https://img.shields.io/badge/runtime-Bun-f9f1e1.svg?logo=bun)
](https://bun.sh/)

---

## Getting Started

```bash
# Install dependencies
bun install

# Start development server
bun run dev
```
