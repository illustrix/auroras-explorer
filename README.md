# Auroras Explorer

A community web tools collection for [Prosperous Universe](https://prosperousuniverse.com/), built to help players explore game data, plan production, and collaborate with others.

> 本项目是基于 [illustrix/auroras-explorer](https://github.com/illustrix/auroras-explorer) 的汉化版本，原项目访问地址：https://explorer.auroras.xyz

## Development

```bash
bun install
bun run dev
```

> 如果没有安装 bun，可以使用 `npm install --legacy-peer-deps` 代替。
