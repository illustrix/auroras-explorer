# 电子与软件

游戏中最复杂的生产链。**44 种电子 + 17 种软件**层层堆叠成深度依赖网络 —— 许多船只组件和高级消耗品离原料有 8–16 个生产步骤。利润丰厚但对新手不友好。如果想卖 CC（计算核心）或 WR（警戒雷达），你就是在承诺建立一个多殖民地的长线供应网。

## 第 1 级：电子小件（10 种）

最小的构建块。便宜、快速、高产。

### 精密锻造厂（FS）—— 金属类小件

| 代号 | 名称 | 配方 | 时间 |
|------|------|------|------|
| **SFK** | 小型紧固件套件 | 1× FE → 16× SFK | 2.9h |
| **MFK** | 中型紧固件套件 | 1× STL → 8× MFK | 2.9h |
| **BCO** | 基础铜线 | 1× CU + 300× PE → 10× BCO | 7.2h |
| **BGC** | 基础金组件 | 1× BGO + 300× PE → 10× BGC | 7.2h |
| **HCC** | 高容量组件 | 300× PE + 1× RGO → 10× HCC | 7.2h |

### 洁净室（CLR）—— 硅类小件

| 代号 | 名称 | 配方 | 时间 |
|------|------|------|------|
| **CAP** | 电容 | 1× FE + 1× SI → 15× CAP | 9.6h |
| **LDI** | 激光二极管 | 1× SI + 1× GAL → 6× LDI | 8.4h |
| **MWF** | 中型晶圆 | 2× SI + 2× AL → 20× MWF | 12h |
| **SWF** | 小型晶圆 | 1× AL + 1× SI → 30× SWF | 12h |
| **TRN** | 晶体管 | 1× SI + 1× AL → 10× TRN | 6h |

> **SFK 和 MFK 是万金油。** 两者用在几十个配方里（紧固件把一切连在一起）。周期只有 2.9h —— 1 个 FS 每天能产几百个。

## 第 2 级：电子部件（11 种）

在 **SCA**（技工）和 **MCA**（移民+技工）中组装。是 PrUn 的"芯片"层。

### 小型组件装配（SCA）

| 代号 | 名称 | 配方 | 时间 |
|------|------|------|------|
| **PCB** | 印刷电路板 | 5× BCO + 10× SWF + 1× BGO + 60× PE → 5× PCB | 6.7h |
| **MPC** | 微处理器 | 10× BCO + 10× SWF + 10× TRN → 10× MPC | 10.1h |
| **RAM** | 随机存取存储 | 6× BCO + 6× CAP + 6× PSS → 6× RAM | 10.1h |
| **ROM** | 只读存储 | 10× BGC + 10× PSS + 10× SI → 10× ROM | 7.2h |
| **SEN** | 传感器 | 10× H + 10× N + 20× TRN → 20× SEN | 7h |
| **TPU** | 张量处理单元 | 4× MWF + 4× CAP + 4× HCC + 4× TRN → 4× TPU | 15.6h |

### 中型组件装配（MCA）

| 代号 | 名称 | 配方 | 时间 |
|------|------|------|------|
| **DIS** | 显示器 | 1× LCR + 3× GL → 1× DIS | 16.1h |
| **CD** | 紧凑显示器 | 1× DIS + 2× BCO → 1× CD | 16.1h（在 ELP 中） |
| **MB** | 主板 | 4× MPC + 4× PSM + 4× SFK + 4× RAM → 4× MB | 7.2h |
| **HD** | 硬盘 | 5× LDI + 5× DCS + 1× RG → 5× HD | 16.1h |
| **FAN** | 风扇 | 8× DCS + 1× AL + 2× SFK → 8× FAN | 4.8h |

### 洁净室（也能做）

| 代号 | 名称 | 配方 | 时间 |
|------|------|------|------|
| **TRA** | 收发器 | 1× SWF + 1× CAP + 1× TRN → 1× TRA | 4.8h |

> **PCB 是电子链的中心枢纽。** 出现在 10+ 个配方中作为输入。如果能廉价生产 PCB（来自金矿丰富星球的低价 BGO 供应），终生受益。

## 第 3 级：电子设备（12 种）

成品硬件。要么被工人直接消耗（SCN、PDA），要么进入系统。

|------|------|------|------|

| **AWF** | 大气水过滤器 | 10× BAC + 10× PE + 1× MAG → 5× AWF | 15.6h |
| **BMF** | 体感框架 | 1× MB + 1× TPU + 1× DCL + 1× FAN → 1× BMF | 16.8h |
| **BWS** | 基础工作站 | 1× DCM + 1× HD + 1× MB → 1× BWS | 12h |
| **HPC** | 高性能计算机 | 1× MB + 1× DCS + 1× CD → 1× HPC | 8.4h |
| **SAR** | 传感器阵列 | 16× SEN + 4× PCB → 1× SAR | 16.1h |

### EDM（移民级设备）

| 代号 | 名称 | 配方 | 时间 |
|------|------|------|------|
| **MHP** | 微型热泵 | 1× TRA + 1× PCB → 1× MHP | 4.8h |
| **RAD** | 收音机 | 1× DCS + 1× POW + 1× SFK → 1× RAD | 4.8h |
| **BID** | 生物反应植入器 | 2× CD + 1× KV + 2× SAR → 2× BID | 7.2h |
| **HOG** | 均质机 | 1× HD + 1× DCS + 1× NG → 1× HOG | 7.2h |

### 制药厂（PHF）

| 代号 | 名称 | 配方 | 时间 |
|------|------|------|------|
| **BSC** | 生物合成舱 | 1× DCM + 1× MFK + 1× SAR → 1× BSC | 21.6h |

| 代号 | 名称 | 配方 | 时间 |

## 小技巧

1. **从 FS + CLR 开始。** 两者都只要移民/技工 —— 不需要工程师。卖 BCO/PCB/SFK 是稳定的中期利基。
2. **金矿供应是 PCB 的瓶颈。** BGO → PCB 每 5× PCB 需要 1× BGO。便宜的 AU 来源是 PCB 生意的基石。
3. **SEN 被低估了。** 每个 SAR 要 16× SEN，而 SAR 进入 5+ 个配方。一个专做 SEN + SAR 的 SCA 能供应整个系统工厂。
4. **软件时间惊人。** MLI 72h、WAI 48h、CC 57.6h。开始终局电子前要按 10+ 天的生产窗口规划。
5. **不要跳过软件链。** 做 CC/WR/FFC/LIS 的时候，同时也要运营 SD/SE/SL。预算 3 个建筑 + 科学家住房。
6. **AAF 是船只级系统的大门。** 40 工程师 + 20 科学家工人。需要成熟的殖民地。有意义的产能至少要 2-3 个 AAF。
7. **PCB > MPC > MB 是主装配链。** 每一层消耗上一层 —— 保持吞吐平衡，否则会有一层堆积。
8. **洁净室（CLR）能跑 6 种配方。** 电子链中最灵活的工种 —— 一个 CLR 可以在 CAP、LDI、MWF、SWF、TRN、TRA 之间切换。
