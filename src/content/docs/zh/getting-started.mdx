# 欢迎来到 Prosperous Universe

**Prosperous Universe (PrUN)** 是一款基于浏览器的多人经济模拟游戏，玩家在其中建立和管理星际企业。你将从星球上开采资源、制造商品、在商品交易所进行交易，并与其他玩家在广阔的星系中合作。

## 核心玩法循环

游戏围绕一个简单但深度丰富的生产链展开：

1. **开采** 星球上的原材料（矿石、气体、水等）
2. **精炼** 原材料为中间产品（金属、化学品、塑料）
3. **制造** 中间产品为高级产品（电子元件、船舶部件、预制件）
4. **交易** 在商品交易所（CX）出售产品获取利润
5. **扩张** 将业务拓展到新的星球和产业

## 你的第一个基地

开始游戏时，你需要在一颗星球上建立第一个基地。关键考虑因素：

- **星球肥沃度** — 决定是否可以从事农业（种植食物、生产有机材料）
- **资源浓度** — 浓度越高，开采速度越快
- **与 CX 的距离** — 距离商品交易所越近，运输成本越低
- **劳动力供应** — 你的建筑需要工人（先驱者、定居者、技术员、工程师、科学家）

## 劳动力等级

| 等级 | 角色 | 典型建筑 |
|------|------|---------|
| 先驱者 (Pioneer) | 基础劳动 | 农场 (FRM)、钻井平台 (RIG)、采集器 (EXT) |
| 定居者 (Settler) | 技能劳动 | 冶炼厂 (SME)、食品加工厂 (FP) |
| 技术员 (Technician) | 技术工作 | 电子工厂 (ELP)、预制件工厂 |
| 工程师 (Engineer) | 高级工作 | 船舶套件工厂 (SKF)、软件开发 |
| 科学家 (Scientist) | 研究工作 | 实验室 (LAB) |

每个等级需要特定的消耗品 —— 先驱者需要基本口粮（RAT）、饮用水（DW）和工作服（OVE），而科学家则需要更多的奢侈品。

## 商品交易所

星系中散布着多个商品交易所（CX），主要包括：

- **AI1** — Antares I
- **CI1/CI2** — Ceres Imperium
- **NC1/NC2** — Nearest Center
- **IC1** — Ion Cluster

> **提示：** 不同交易所之间的价格差异很大。在一个 CX 便宜的材料在另一个 CX 可能很贵 —— 这创造了套利机会！
