# 消耗品

工人不会免费给你干活 —— **29 种消耗品**负责让他们吃饱、穿暖、健康、开心。缺少任何一种必需消耗品，对应等级的建筑就会减速或停工。自给自足通常比从 CX 买更省钱；给**其他**玩家的工人供货则是稳定的长线生意。

## 奢侈消耗品（10 种）

| 代号 | 名称 | 建筑 | 配方 |
|------|------|------|------|
| **COF** | 咖啡 | FP | 1× CAF + 3× DW → 3× COF (7.2h) |
| **KOM** | 康普茶 | FER | 4× DW + 1× AMM + 1× HER → 6× KOM (8.4h) |
| **ALE** | 麦酒 | FER | 1× HOP + 1× AMM + 3× DW + 2× GRN → 6× ALE (15.4h) |
| **GIN** | 金酒 | FER | 1× ES + 2× GRN + 2× DW + 1× AMM → 4× GIN (15.6h) |
| **WIN** | 葡萄酒 | FER | 15× GRA + 15× REA + 20× DW + 1× AMM → 10× WIN (19.2h) |
| **PWO** | 动力工服 | BMP | 50× PG + 1× COT → 20× PWO (7.2h) |
| **REP** | 维修包 | BMP | 1× SFK + 1× MFK + 10× INS → 4× REP (6h) |
| **SC** | 智能芯片 | CHP | 1× MG + 1× TA + 2× CA → 20× SC (15.6h) |
| **VG** | 生命凝胶 | IVP | 2× VIT + 2× BAC + 4× REA + 2× BL → 20× VG (24h) |
| **NST** | 益智药 | LAB | 1× ES + 1× THF + 1× ALG → 4× NST (12h) |

> **酒厂规律：** ALE、GIN、WIN、KOM 都需要 **AMM**（氨）作为发酵剂。建个 CHP 或进口 AMM —— 别低估这项。四种都需要 DW，所以酒厂星球通常自己运营一个 FP 来出水。

## 消耗品捆包（5 种）

捆包是在 **PAC**（包装）建筑中制作的**预打包**消耗品组合。主要用于玩家间供货合同 —— 一个 COLONY 消耗方案 = 每天一个捆包。

| 代号 | 用途 | 配方 |
|------|------|------|
| **PBU** | 先锋 | 50× PE + 5× COF + 40× DW + 40× RAT + 5× OVE + 2× PWO → 1× PBU (4.8h) |
| **SBU** | 移民 | 60× RAT + 100× PE + 10× KOM + 50× DW + 5× EXO + 5× PT + 2× REP → 1× SBU (5.3h) |
| **TBU** | 技工 | 70× RAT + 1× PSS + 1× SCN + 1× SC + 40× PE + 5× HMS + 75× DW + 5× MED + 10× ALE → 1× TBU (6h) |
| **EBU** | 工程师 | 4× PSS + 70× FIM + 1× PDA + 10× GIN + 160× PE + 5× MED + 2× HSS + 100× DW + 2× VG → 1× EBU (6.7h) |
| **CBU** | 科学家 | 10× WIN + 280× PE + 5× MED + 2× LC + 1× WS + 100× DW + 70× MEA + 8× PSS + 1× NST → 1× CBU (7.2h) |

> **PE（聚乙烯）是包装材料。** 每个捆包要用 50–280 PE —— 通常是最大的单项成本。

## 小技巧

1. **基础消耗品优先。** 缺奢侈品效率下降 10–20%。缺基础品生产直接停。永远优先保证基础品。
2. **水是万能的。** 每个等级都需要 DW。可靠的水供应（本地开采 H2O 或进口）是大多数殖民地的头号瓶颈。
3. **RAT 撑起 3 个等级。** 先锋、移民、技工都吃 RAT —— 是游戏中产量最大的消耗品。早期就要多建几个 FP。
4. **工程师和科学家吃 FIM/MEA。** 这些是从 RAT → FIM → MEA 加工而来。升级工人等级前，先规划好食物链。
5. **奢侈品按工人数提升效率 —— 先上便宜的。** 先锋的 COF 最便宜；移民的 KOM 最便宜。从第一天就至少供应一种奢侈品。
6. **自给自足 vs 从 CX 购买。** 算一下每天成本：100 先锋每天吃 4 RAT × 3 代币 CX 价 + 4 DW × 3 代币 ≈ 20 代币/天。和本地种 2 FRM + 1 FP 比较一下。
7. **捆包合同是稳定收入。** 如果有富余产能，卖长期 PBU/SBU 合同比在 CX 上追涨杀跌稳定得多。
