# 物品分类

Prosperous Universe 拥有超过 **370 种物品**，分布在 **35 个分类**中。理解这些分类对于规划你的生产链至关重要。

## 原材料

这些是一切的基础 —— 直接从星球上开采，无法通过制造获得。

### 矿石 (8 种)

金属的主要来源。使用**采集器 (EXT)** 开采。

| 代码 | 名称 | 冶炼产物 |
|------|------|---------|
| FEO | 铁矿石 | FE (铁) |
| ALO | 铝矿石 | AL (铝) |
| CUO | 铜矿石 | CU (铜) |
| SIO | 硅矿石 | SI (硅) |
| AUO | 金矿石 | AU (金) |
| LIO | 锂矿石 | LI (锂) |
| TIO | 钛矿石 | TI (钛) |
| REO | 稀土矿石 | 各种稀有金属 |

### 气体 (10 种)

使用**收集器 (COL)** 采集。对化学工业和燃料生产至关重要。

主要气体包括 **H**（氢气）、**O**（氧气）、**N**（氮气）、**HE**（氦气）、**HE3**（氦-3）和 **F**（氟气）。

### 矿物 (14 种)

从星球表面开采。应用于从建筑到电子的众多行业。

重要矿物：**LST**（石灰石）、**HAL**（岩盐）、**BER**（铍晶体）。

## 中间材料

### 金属 (10 种)

在**冶炼厂 (SME)** 或**电弧炉 (AML)** 中由矿石冶炼而成。制造业的支柱。

- **AL**（铝）、**FE**（铁）、**CU**（铜）—— 基础金属，需求量大
- **STL**（钢）—— 需要铁 + 碳
- **TI**（钛）、**W**（钨）—— 高端生产所需的高级金属

### 化学品 (21 种)

在**化工厂 (CHP)** 和**实验室 (LAB)** 中生产。用途广泛。

### 塑料 (8 种)

在**聚合物工厂 (POL)** 中由碳氢化合物制成。用于建筑和消费品。

## 终端产品

### 船舶组件

游戏中最复杂的产品。船舶引擎如 **VOE**（涡旋引擎）位于生产链的**第 16 层** —— 从原材料到最终产品需要 16 个连续加工步骤！

### 消耗品包

为劳动力消耗预先打包的套装：

| 套装 | 适用人群 | 主要内容 |
|------|---------|---------|
| PBU | 先驱者 | RAT、DW、基础物资 |
| SBU | 定居者 | MEA、更多品种 |
| TBU | 技术员 | 更高品质的食物 |
| CBU | 科学家 | 奢侈品 |
| EBU | 工程师 | 优质物资 |

### 建筑预制件

星球建筑的建造模块。按劳动力等级分层。

## 物品属性

每种物品都有两个关键的物理属性：

- **重量**（吨）—— 影响货舱的质量容量
- **体积**（m³）—— 影响货舱的空间容量

> **重要：** 运输货物时，你受到**重量和体积双重限制**。有些材料是"重型"的（高重量、低体积），如矿石；而其他材料则是"蓬松"的（低重量、高体积），如某些气体。
