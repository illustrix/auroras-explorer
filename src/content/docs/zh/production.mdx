# 生产与配方

生产是 Prosperous Universe 的核心。每一件制造品都是通过将特定的输入材料放入建筑中，等待配方完成来制造的。

## 配方的工作原理

一个配方定义了：

- **建筑** — 哪种建筑类型执行此配方（如 SME、FP、CHP）
- **输入** — 消耗的材料（如 1× FEO）
- **输出** — 生产的材料（如 1× FE）
- **持续时间** — 生产周期需要多长时间

例如，冶炼铁：

> **SME:** 1× FEO → 1× FE（4.8 小时）

## 配方统计

游戏包含 **389 个配方**，涉及 **49 种建筑**。以下是概览：

| 指标 | 数值 |
|------|------|
| 配方总数 | 389 |
| 建筑类型 | 49 |
| 配方最多的建筑 | FP (21)、BMP (20)、UPF (19) |
| 最快配方 | ~2.4 小时 |
| 最慢配方 | 360 小时 (HAM) |

## 生产链深度

物品在生产链中处于不同的"深度"：

- **深度 0** — 原材料（44 种）：矿石、气体、水等
- **深度 1–4** — 基础加工品：金属、简单化学品
- **深度 5–7** — 中间产品：电子元件、建筑零件（约 150 种）
- **深度 8–12** — 高级产品：船舶组件、软件
- **深度 13–16** — 终局产品：涡旋引擎、星门段

最深层的物品（**GWS** 和 **VOE**，深度 16）需要原材料经过 16 个连续加工步骤才能成为最终产品。

## 食品生产：RAT 配方

游戏早期最重要的产品之一是 **RAT（口粮）**。所有 RAT 配方遵循统一规律：

> **从三组材料中各选一种 → 10× RAT（6 小时）**

| 碳水化合物 | 蛋白质 | 脂肪 |
|-----------|--------|------|
| GRN (谷物) | ALG (藻类) | NUT (坚果) |
| MAI (玉米) | BEA (豆类) | VEG (蔬菜) |
| MUS (蘑菇) | — | — |

最经济的组合只使用农场产品：**MAI + BEA + NUT** —— 全部在基础农场（FRM）中仅用水即可种植。

## 生产规划建议

1. **平衡劳动力** — 不要过度建造某一层级。每栋建筑都需要工人得到喂养和住房。
2. **考虑运输成本** — 从 CX 购买中间产品可能比自己生产更便宜。
3. **关注市场** — 价格会波动。今天盈利的配方明天可能就是亏损的。
4. **规划运行时间** — 当输入材料耗尽时，建筑不会生产任何东西。保持关键材料的缓冲库存。
