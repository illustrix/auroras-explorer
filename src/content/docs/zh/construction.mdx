# 建筑材料

你在星球上盖的每个建筑都需要**预制件**，而预制件又消耗**零件**和**材料**。这条链没有 CX 上的终局产品；它纯粹是为支撑基地扩张而存在的。但规模很大：**44 项**分布在预制件、零件、材料三类，由 4 级预制件工厂主导。

## 预制件等级系统

预制件分 4 个等级，对应它们所建的 4 个工人等级住房：

| 等级 | 工厂 | 前缀 | 工人 | 建造 |
|------|------|------|------|------|
| **基础** | PP1（预制件厂 1） | `B-` | 80 先锋 | 先锋住房（HB1, HBB） |
| **轻型** | PP2 | `L-` | 25 先锋 + 25 移民 | 移民/技工住房（HB2, HB3） |
| **加固** | PP3 | `R-` | 20 移民 + 40 技工 | 工程师住房（HB4）、专用 |
| **高级** | PP4 | `A-` | 40 技工 + 30 工程师 | 科学家住房（HB5）、豪华住房（HBL, HBM） |

每级都做 4 种标准预制件 —— **隔板（BH）**、**穹顶（DE）**、**支架（SE）**、**塔（TA）** —— 用对应字母前缀命名。

## 标准预制件（16 种）

### 基础等级（PP1）—— 先锋工人

| 代号 | 名称 | 配方 | 时间 |
|------|------|------|------|
| **BBH** | 基础隔板 | 1× LST + 2× FE → 1× BBH | 7.9h |
| **BDE** | 基础穹顶 | 150× PE → 1× BDE | 7.2h |
| **BSE** | 基础支架 | 2× LST + 1× FE → 1× BSE | 6h |
| **BTA** | 基础塔 | 50× PE + 1× FE → 1× BTA | 3.6h |

### 轻型等级（PP2）—— 先锋 + 移民

| 代号 | 名称 | 配方 | 时间 |
|------|------|------|------|
| **LBH** | 轻型隔板 | 3× AL + 35× PE → 1× LBH | 6h |
| **LDE** | 轻型穹顶 | 3× AL + 1× NL → 1× LDE | 6h |
| **LSE** | 轻型支架 | 3× AL + 120× PG → 1× LSE | 6h |
| **LTA** | 轻型塔 | 5× GL + 1× AL → 1× LTA | 6h |

### 加固等级（PP3）—— 移民 + 技工

| 代号 | 名称 | 配方 | 时间 |
|------|------|------|------|
| **RBH** | 加固隔板 | 1× BBH + 50× EPO + 1× STL → 1× RBH | 9.6h |
| **RDE** | 加固穹顶 | 1× LDE + 100× EPO + 1× KV → 2× RDE | 16.8h |
| **RSE** | 加固支架 | 2× BSE + 1× STL + 225× EPO → 2× RSE | 16.8h |
| **RTA** | 加固塔 | 1× LTA + 6× RG → 1× RTA | 9.6h |

### 高级等级（PP4）—— 技工 + 工程师

| 代号 | 名称 | 配方 | 时间 |
|------|------|------|------|
| **ABH** | 高级隔板 | 2× RBH + 125× NR → 2× ABH | 16.8h |
| **ADE** | 高级穹顶 | 2× KV + 2× LDE → 2× ADE | 14.4h |
| **ASE** | 高级支架 | 1× RSE + 2× TI → 1× ASE | 13.2h |
| **ATA** | 高级塔 | 1× RTA + 1× NG → 1× ATA | 13.2h |

> **升级链：** 很多高级预制件**消耗低级的**（RBH 用 BBH，ABH 用 RBH）。可以垂直生产 —— 一个星球上 PP1→PP3→PP4 生产 ABH。

## 专用预制件（4 种）

非分级预制件，用于特定目的：

| 代号 | 名称 | 工厂 | 配方 | 用途 |
|------|------|------|------|------|
| **HSE** | 重型支架 | PP3 | 1× TCS + 2× LSE → 2× HSE | 重载结构（TOR 等） |
| **MGC** | 电磁线圈 | PP3 | 1× LDE + 1× MAG → 1× MGC | 电磁类建筑 |
| **PSH** | 塑料屏蔽 | PP3 | 250× NFI + 1× TI → 1× PSH | 反应堆屏蔽 |
| **RSH** | 辐射屏蔽 | PP4 | 1× TA + 1× STL + 1× LST → 1× RSH | 核类建筑 |
| **TSH** | 热屏蔽 | PP4 | 2× THP + 2× LBH + 150× PE → 1× TSH | 高温建筑 |

## 住房成本（各级住房所需预制件）

真正的问题：建每种住房需要多少预制件？

| 住房 | 用于 | 预制件清单 |
|------|------|-----------|
| **HB1** | 先锋 | 4× BBH + 2× BDE + 2× BSE + 1× BTA |
| **HBB** | 先锋（兵营） | 4× BBH + 4× BDE + 2× LSE + 2× LTA |
| **HB2** | 移民 | 2× BBH + 2× BDE + 2× BSE + 2× BTA + 2× TRU |
| **HB3** | 技工 | 4× LBH + 4× LDE + 4× LSE + 8× LTA |
| **HBC** | 技工（公社） | 4× LBH + 2× LDE + 2× RSE + 4× RTA + 2× TRU |
| **HB4** | 工程师 | 6× RBH + 6× RDE + 4× RSE + 4× RTA |
| **HBM** | 工程师（管理者） | 8× RBH + 8× RDE + 6× ASE + 4× ATA |
| **HB5** | 科学家 | 6× ABH + 6× ADE + 4× ASE + 4× ATA |
| **HBL** | 科学家（豪华） | 8× ABH + 8× ADE + 6× ASE + 6× ATA |

> **提示：** HB1 用 9 个预制件；HBL 用 28 个。高级殖民地住房要**约 3 倍的建筑成本**。这是为什么后期基地扩张变慢的原因之一。

## 小技巧

1. **PP1+PP2 加先锋移民工人是很稳的起步生意。** BBH/LBH/LDE 等产品可以卖给区域内每个扩张中的殖民地。
2. **EPO 是游戏的胶水。** 每个 R 级和 A 级预制件都用大量 EPO。早中期通常会专门建一个 POL 做 EPO。
3. **PE 需求量巨大。** 单个 BDE 用 150 PE。做塑料生意的话，PE 是出货主力。
4. **RBH 和 RSE 用 50–225 EPO。** EPO 成本在这些配方中占主导 —— 定价时要考虑进去。
5. **科学家住房成本是先锋的 3 倍。** ABH 是 BBH→RBH→ABH 的垂直堆叠。要算上 3 倍的供应链时间。
6. **屏蔽类（RSH/TSH/PSH）虽然小众但必不可少。** 只有特定建筑需要，但需要的时候没有替代品。
7. **AIR 之类的复杂零件对大多数玩家来说只能合同购买。** 从零造 AIR 需要 5 条不同的专业链 —— 几乎总是买比做便宜。
